コーディング効率化ツール「Coding Support」

以下の悩みを解決する目的で作りました。

  • デザインと全く同じ見た目でコーディングしたい
  • スクロールアニメーション開発時のリロード回数を減らしたい
また、開発にあたって以下の技術を使用しました。
  • Javascript
  • Local Strage(Cookieの上位互換)

「Coding Support」で出来ること

  • html cssマークアップを効率化出来る
  • フォトショップのようなガイドラインをブラウザで表示出来る
  • jpg等の一枚画像を重ねて表示できる（カバー画像）
  • カバー画像は任意の値で透過させたり、フィルターをかけたり、または上部からどれだけズラすか設定できる
  • スクロールアニメーションや、ローディングアニメーションを使用するサイトでは、ロードさせなくてもボタンクリックでclassを付けたり外したり出来る

画像URLや設定値などは「localstrage（cookieの上位互換）」に保存される為、値は保持されます。
「リロードしたら設定が飛ぶ」というような事はありません。

導入方法

以下の２通りがあります。

  • ①URL（https://maru-code.com/tools/cs/main.js）をscriptタグでhtmlファイルに直に読み込む。
  • ②URL（https://maru-code.com/tools/cs/main.js）を、GoogleChrome拡張機能「ScriptAutoRunner」の中で読み込む（推奨）

①の例：下記のコードをhtmlファイル閉じbodyタグ直前等に貼り付けます。

<script src="https://maru-code.com/tools/cs/main.js"></script>


本当は「Chrome拡張機能」として登録したかったところですが、技術的に断念しました。

私は②のScriptAutoRunnerを使用しています。（ScriptAutoRunnerとは、任意のjsを気軽に実行できる拡張機能です）

ちなみに①「htmlに直読み込み」をすると、スマホ実機でも確認できるというメリットもあります。ただし、納品前のコード消去は必須です。

「Coding Support」の使い方

導入に成功したら、画面右側に以下のボタンが表示されます。

当記事ではこれを「コントロールボタン」と呼びます。

コントロールボタン一番下の「H」を押すと、メインの「コントロールパネル」が表示されます。（または非表示になります）

まずは「コントロールボタン」を解説します。

「コントロールボタン」の説明

  • 「G」→ ガイドラインの略。クリックでガイドラインの表示非表示を切り替えます。
  • 「P」→ Pictureの略。クリックでカバー画像の表示非表示を切り替えます。
  • 「A」→ Animetionの略。クリックでアニメーション用class等のclassを付けたり外したりします。
  • 「H」→ Hideの略。クリックでコントロールパネルの表示非表示を切り替えます。

「コントロールパネル」の説明

「ガイドライン」については、inputの「placeholder（初期値）」に詳細を書いているのでここでは省きます。

「カバー画像」の使い方

  • 参考用画像「https://maru-code.com/tools/cs/images/image03.jpg」のURLを、コントロールパネル「画像 URL」に貼り付けます。
    すると当ページのデザイン画像が表示されます。
  • コントロールボタン「P」で、カバー画像の表示非表示を切り替えます。

画像 URLはhttp形式で取得する必要があるので、ローカルサーバーやタスクランナーツールを用いるか、開発用サーバーに一時アップすると作業しやすいかと思います。

一応、「カバー画像」の「ページ上部からの距離」「透過度」「フィルター」だけ簡単に解説すると、それぞれcssの以下の部分に当たります。

  • ページ上部からの距離 → top: ココに入る px;
  • 透過度(css) → opacity: ココに入る;
  • フィルター(css) → filter: ココに入る;

その為、css形式で書き込む必要があります。

filterは、「今見ているのが本物かカバー画像か分からなくなる」という事故を防ぐ為に追加しました。明度や彩度を変えたりすると使いやすいです。 

filterの例：saturate(80%) sepia(100%)

動きのあるサイトでよく見られる「スクロールしたら、“js-toggle”というclassがある要素に、“display”というclassが付与される」という挙動を例として説明します。
トリガーclass
「js-toggle」と入力します。
トグルclass
「display」と入力します。

こうすることで、コントロールボタンの「A」をクリックすると、「.js-toggle要素」に対して「.display」の付け外しが発動します。

※実務ではそれぞれ任意のclass名に変更してください。
