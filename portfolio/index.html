2014年からweb制作を始め、複数のコーポレートサイト制作、ECサイト運営等に携わってきました。

現時点で、HTML/CSSに関しては習熟しておりますが、プログラミングはまだ初級レベルである為、現在、生のJavascriptを使い、イチからプログラムを書いてみたり、Vue.jsの勉強を行っております。

得意なこと

作業の効率化が好きで、コーディングの作業速度アップに関することは積極的に取り入れております。
（エディタのセッティングやプラグイン導入等）

また、javascriptの練習も兼ねて、自分の作業環境や日常生活を良くするための簡易的なツールも制作しています。

自作したツール

デザイン通りにコーディングできるようになるコーディング補助js
GoogleChromeの拡張機能でも似たようなものがありますが、より自分が作業しやすいよう、「アニメ―ション用発火処理のtoggle機能」等も追加しました。

必要アイテムをメモするツール
いわゆる「ToDoリスト」と言われるツールです。とあるゲームをプレイ中に、「こんなツールがあったら良いな」と思い自作しました。

スキルセット

● html/css
レスポンシブ、サイト軽量化、SEOを意識したコーディングが可能です。
pugやscssを使用したコーディングをメインに行っており、制作会社の頃はgulpを使ってコンパイルを行っておりました。

● javascript
DOM操作に関する事であれば概ね自己完遂出来る程度の技術力です。
データ操作については実務経験がなく、AJAXやJSフレームワーク（Vue.js）、Local Strage等も、自己学習内での制作経験のみです。

● Wordpress
ゼロからのTheme構築が可能。
カスタム投稿、カスタムフィールド機能（プラグイン使用）の追加、呼び出し等の経験があります。

● デザインソフト
Photoshop → 画像加工、スライス、バナーデザイン等
Illustrator → アイコン作成、スライス
Lightroom → 画像加工
その他、Adobe XDやFigma等のスライス経験があります。

● デザインスキル
補佐的な作業、例えば「納品はPC版デザインのみで、SP版を適切に解釈し、一部画像加工しながらコーディングする」等が出来ます。
ページデザインは力不足ですが、一部経験があります。
あおやぎ様 花の会ページ（https://www.aoyagi-sougi.co.jp/flower_member/）は私がデザインしました。
また、バナーや商品画像のような小規模な制作物であれば50件以上程度の経験があります。
